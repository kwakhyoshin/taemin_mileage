# 마일리(mile.ly) 업그레이드 로드맵

> 태민이 마일리지 PWA의 향후 업그레이드 계획. 이 문서는 계획서이며 실제 변경 이력은 `dev_guide.md`에 기록한다.

## Phase 1. 게이미피케이션 심화 (재미·동기부여)

- **콜렉션 도감**: 챌린지 곤충 / 레벨 이펙트 / 캐릭터 등을 도감 형태로 모으고, 각 항목별 최초 획득일·획득 조건·짧은 설명 표시.
- **시즌 패스 / 이벤트**: 매월 한정 챌린지와 시즌 한정 보상(테마, 곤충, 배경 등). 시즌 종료 시 명예의 전당에 기록.
- **마일리 캐릭터 진화**: 누적 마일리지·연속 사용 일수에 따라 마일리 캐릭터가 단계별로 진화(아기 → 청년 → 어른 → 전설 등). 진화 시 축하 애니메이션.
- **연속일수(streak) 보호 아이템**: 깜빡하고 하루를 놓쳐도 streak이 깨지지 않도록 하는 "보호막" 아이템. 마일리지로 구매 또는 주간 미션 보상.

## Phase 2. 가족 협업 기능 강화

- **가족 챌린지**: 부모·자녀가 함께 달성하는 협동 챌린지 (예: "이번 주 가족 모두 책 3권 읽기"). 달성 시 가족 전체 보상.
- **응원 스티커·이모지 반응**: 활동 카드에 부모/형제자매가 하트·이모지·짧은 응원 메시지를 남길 수 있는 기능.
- **주간 가족 리뷰**: 매주 일요일 자동 생성되는 한 주 하이라이트 (최고 활동, 새 뱃지, 칭찬 메시지 모음 등).
- **형제자매 비교·협력 모드**: 멀티 자녀 가구를 위한 친근한 랭킹·협력 모드 (경쟁이 아닌 응원 중심).

## Phase 3. AI 챗봇 고도화

- **주간 코치 리포트**: 활동 로그를 분석해 개인화된 칭찬·다음 주 추천 챌린지·관심사 변화 등을 AI가 작성.
- **음성 입력**: 활동 기록을 마이크로 빠르게 등록 (특히 어린이 친화적).
- **이미지 인증**: 사진을 찍으면 AI가 활동을 자동 인식해 매칭 (예: 책 표지 → 독서 활동 자동 등록).
- **부모 대신 답변**: 부모가 답을 미루고 있을 때 AI가 톤을 학습해 임시 응원 메시지 제안.

## Phase 4. 기반 안정화·성장 준비

- **단일 25k줄 HTML 모듈화**: Vite 기반 빌드 도입, 컴포넌트·기능 단위 파일 분리. 유지보수성과 협업 효율 향상.
- **Firestore 백업/복원 자동화**: 정기 스냅샷 + 원클릭 복원. 데이터 사고 대비.
- **에러 모니터링**: Sentry 등을 도입해 실사용자 환경의 JS 오류·렌더링 이슈를 자동 수집.
- **PRO 등급 결제 연동**: TossPayments / Stripe 등 결제 모듈 연결, 무료/유료 기능 분기.
- **다국어(i18n) 기반**: 텍스트 분리·언어 전환 인프라 구축 (영문/일문 등 확장 대비).

## 추천 우선순위

**Phase 1 → Phase 2 → Phase 3 → Phase 4**

게이미피케이션·가족 협업으로 사용자 몰입도를 먼저 끌어올리고, AI 고도화로 차별화한 뒤, 마지막으로 기반을 안정화해 본격 성장 단계로 진입한다.
