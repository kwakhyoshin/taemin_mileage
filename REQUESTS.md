# 요청사항 추적 (Request Tracker)

> 사용자 요청사항의 개발기/운영기 반영 현황을 기록한다.
> 새 요청이 들어오면 반드시 이 문서에 추가하고, 배포 시 상태를 업데이트할 것.

## 상태 범례

| 상태 | 의미 |
|------|------|
| 대기 | 아직 개발 시작 안 함 |
| 개발중 | 개발기에서 작업 중 |
| 개발완료 | 개발기 반영 완료, 운영 미반영 |
| 운영완료 | 개발기 + 운영기 모두 반영 완료 |
| 설계중 | 설계 문서 작성 단계 (구현 전) |
| 보류 | 일시 보류 |
